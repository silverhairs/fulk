# fulk
A simple command-line tool to bulk rename all the files in your current folder.

<img src="/demo.gif">

## Requirements:
- Python 3.6^

## Installation:
```
$ git clone https://github.com/silverhairs/fulk.git
$ cd fulk
$ pip3 install --editable .
```

## Usage

`$ fulk -p <files_prefix>` or `$ fulk --prefix <files_prefix>`
